## Description

Qooper Survey Form App
Created By: Kay Singian.

Written in Typescript using NestJS + Fastify and Prisma.
Testing using Jest
Documentation using Swagger

Deployment Link
https://qooper-survey-84b6be2195c9.herokuapp.com/

Documentation
https://qooper-survey-84b6be2195c9.herokuapp.com/api/

## Installation

```bash
$ npm install
```

## Database

Migrate database by running

```bash
$ npx prisma migrate dev --name init
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
